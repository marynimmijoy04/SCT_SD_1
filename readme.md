# Temperature Converter (C++)

A simple C++ program that converts temperatures between Celsius, Fahrenheit, and Kelvin scales using a menu-driven interface.

## Features
- Celsius to Fahrenheit
- Celsius to Kelvin
- Fahrenheit to Celsius
- Fahrenheit to Kelvin
- Kelvin to Celsius
- Kelvin to Fahrenheit

## How to Run
1. Compile the program using a C++ compiler.
2. Run the executable file.
3. Select a conversion option from the menu.
4. Enter the temperature value.
5. View the converted result.

  This Temperature Converter is a simple C++ project that enables users to convert temperature values between Celsius, Fahrenheit, and Kelvin scales. The program uses a menu-driven interface, allowing users to select the desired conversion type and enter a temperature value. It then performs the appropriate calculation and displays the converted result. This project demonstrates fundamental C++ concepts such as user input, output, conditional logic, switch statements, and arithmetic operations, making it an excellent beginner-level programming project.
